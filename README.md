# OpenAPI example

OpenAPI sample code for Node.js and Rust

## How to use

1. visit <https://beta.openai.com/account/api-keys>
2. create new key and copy secret key
3. create .env
4. input `OPENAI_API_KEY=<secret_key>`

### Node.js

edit index.mjs and comment out

```shell
cd node
npm start
```
